# To setup the project 
## First clone the entire repository 
### git clone https://github.com/idevelop4u/web_app_screening_task_fossee.git

# For the web frontend -> 

### cd web_client
### npm install 
### npm run dev 

# For the backend -> 

### pip install -r requirements.txt

or 

### pip install (package_name)
#### // refer the requirements.txt file in backend folder for package names
### cd backend
### python manage.py runserver

# For the desktop frontend ->

### pip install -r requirements.txt

or 

### pip install (package_name)
#### // refer the requirements.txt file in desktop folder for package names
### cd desktop 
### python window.py 

## .env files are already pre-added
